Le code a été développé en utilisant Racket, un dialecte du langage Scheme, et Visual Studio Code (VS Code) comme éditeur de texte, avec l'extension Racket pour la coloration syntaxique. Pour exécuter le code, il suffit d'installer Racket, d'ouvrir le fichier .scm dans VS Code, puis d'exécuter la commande suivante dans le terminal : racket projectMovie.scm. Cela permet de compiler et d'exécuter le programme.
